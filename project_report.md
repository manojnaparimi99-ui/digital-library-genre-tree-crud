# Digital Library: Book Genre Hierarchy Tree (CRUD)

## Abstract
This project presents the implementation of a Digital Library system using a tree data structure in the C programming language. The system organizes book genres hierarchically and supports CRUD operations (Create, Read, Update, Delete). Dynamic memory allocation and modular programming techniques are used to efficiently manage hierarchical data.

## Keywords
Digital Library, Tree Data Structure, CRUD, C Programming, Hierarchical Data

---

## 1. Introduction
In modern digital systems, organizing data efficiently is crucial. Libraries require structured categorization of books into genres and sub-genres. A tree data structure is ideal for representing hierarchical relationships.

This project implements a Digital Library using a general tree structure in C.

---

## 2. Objectives
- Implement tree data structure in C
- Perform CRUD operations
- Use dynamic memory allocation
- Develop a menu-driven program

---

## 3. Scope of the Project
The project focuses on managing genres and sub-genres. It does not include full book details but can be extended in the future.

---

## 4. Literature Review
Tree data structures are widely used in file systems, XML documents, and hierarchical databases. They provide efficient searching and structured organization.

---

## 5. Methodology
- Use struct to define nodes
- Use pointers for hierarchy
- Apply recursion for traversal

Steps:
1. Create root node
2. Add child nodes
3. Traverse using DFS
4. Perform CRUD operations
